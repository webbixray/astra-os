# Phase 0.2: User Personas & Target Audience

## Primary Personas

### Persona 1: "Sarah" — Head of Marketing (Mid-Market / Enterprise)

| Attribute | Detail |
|---|---|
| **Role** | VP / Director of Marketing at 50-500 person company |
| **Age** | 35-50 |
| **Tech Level** | Moderate — uses HubSpot, Canva, Slack, Notion daily |
| **Pain** | Juggling 8+ tools; no unified view of campaigns; spending 40% of time on coordination instead of strategy |
| **Desire** | One platform where strategy → execution → analysis flows without context switching |
| **AI Comfort** | Wants AI to execute, not just suggest |
| **Why ASTRA** | "I need an AI marketing department, not more software tools." |

### Persona 2: "Marcus" — Marketing Operations Manager

| Attribute | Detail |
|---|---|
| **Role** | Marketing Ops / RevOps at scaling company |
| **Age** | 28-40 |
| **Tech Level** | High — builds workflows, manages martech stack, knows APIs |
| **Pain** | Spends 50% of time stitching tools together; fragile automations break monthly; no audit trail |
| **Desire** | Visual workflow builder with AI, version control, governance, and monitoring |
| **AI Comfort** | Wants programmable AI agents he can configure |
| **Why ASTRA** | "I want to build automations that learn, not just execute." |

### Persona 3: "Elena" — Content Marketing Director

| Attribute | Detail |
|---|---|
| **Role** | Content / Creative Director |
| **Age** | 30-45 |
| **Tech Level** | Low-Moderate — uses Canva, Google Docs, CMS |
| **Pain** | Writing briefs, reviewing drafts, managing writers, maintaining brand voice across 50+ pieces/month |
| **Desire** | AI that knows brand voice, generates drafts, manages content calendar, learns from performance |
| **AI Comfort** | Wants AI to generate; wants human to approve |
| **Why ASTRA** | "I want AI that remembers our brand voice from project to project." |

| Attribute | Detail |
|---|---|

## Target Market Segmentation

| Segment | Size | Entry Point | Revenue Model |
|---|---|---|---|
| **SMB** (1-50 employees) | 12M companies globally | Self-serve, no-code | Subscription ($50-200/mo) |
| **Mid-Market** (50-500) | 500K companies | Guided onboarding | Subscription ($500-2,500/mo) |
| **Enterprise** (500+) | 50K companies | Sales-led, custom | Enterprise ($2,500-20,000+/mo) |
| **Agencies** | 100K firms | White-label, multi-tenant | Platform fee + rev share |

## User Roles within ASTRA OS

| Role | Permissions | Primary Interface |
|---|---|---|
| **Owner** | Full access, billing, team management | AI Command Center |
| **Admin** | Full access except billing | AI Command Center |
| **Marketing Manager** | Campaign management, approval authority | Campaign Dashboard |
| **Content Creator** | Content studio, brand library | Content Studio |
| **Analyst** | Analytics, reports, dashboards | Analytics Hub |
| **Viewer** | Read-only, dashboards | Analytics Hub |
